개인정보처리방침

구구단게임 (Global Gugudan Battle) · 앱 개발자: 흥권

본 개인정보처리방침은 구구단게임 (Global Gugudan Battle) 앱(이하 "앱")이 수집·이용·보관하는 개인정보에 관한 사항을 안내합니다.

1. 수집하는 정보

  • 닉네임: 사용자가 직접 입력하는 게임 닉네임 (실명 아님)
  • 학교명: 학교 대항전 참여 시 사용자가 직접 입력 (선택사항)
  • 게임 점수·기록: 구구단 풀이 시간, 정답 수, 난이도 등
  • Google 계정 정보: 로그인 시 표시 이름, 프로필 사진 URL, 이메일 주소 (Google 로그인 선택 시)
  • 기기 설정: 언어, 효과음 온/오프 (기기 내 로컬 저장)

2. 정보 수집·이용 목적

  • 글로벌 랭킹 및 학교 대항전 기록 표시
  • 닉네임 중복 방지 및 본인 기록 식별
  • 게임 개인 최고 기록 저장
  • 앱 서비스 품질 개선

3. 정보 저장 위치

  • 기기 내 로컬: 닉네임, 최고기록, 설정값 (SharedPreferences)
  • Firebase Firestore (Google Cloud): 랭킹 등록 시 닉네임·점수·타임스탬프
  • Firebase Auth (Google): Google 로그인 선택 시 UID·표시 이름·프로필 사진

4. 제3자 서비스

본 앱은 아래 제3자 서비스를 사용합니다. 각 서비스의 개인정보처리방침을 참고하세요.

  • Google Firebase (Firestore, Authentication, Hosting) — privacy policy
  • Google AdSense (광고) — privacy policy

5. 광고

비프리미엄 사용자에게 Google AdSense 배너 광고가 표시됩니다. AdSense는 쿠키 및 광고 ID를 사용하여 맞춤 광고를 제공할 수 있습니다. 앱 내 프리미엄 구독 시 광고가 제거됩니다.

6. 어린이 개인정보 보호

본 앱은 만 13세 미만 어린이를 대상으로 개인정보를 의도적으로 수집하지 않습니다. 보호자가 자녀의 정보 수집 사실을 인지한 경우 아래 연락처로 알려주세요.

7. 정보 보관 기간 및 삭제

  • 로컬 데이터: 앱 삭제 시 즉시 삭제
  • Firebase 랭킹 데이터: 삭제 요청 시 30일 이내 삭제
  • 앱 내 설정 → "기록 초기화"로 로컬 데이터 직접 삭제 가능

8. 사용자 권리

개인정보 열람, 수정, 삭제를 요청하실 수 있습니다. 아래 이메일로 문의해 주세요.

9. 방침 변경
